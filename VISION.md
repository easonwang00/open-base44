# Vision

## What NativeBot IS

- A CLI tool that builds mobile apps from natural language
- Powered by Claude AI (Anthropic's Agent SDK)
- Local-first: your machine, your API key, your code
- Opinionated: Expo React Native for maximum cross-platform reach
- Simple: one install command, one entry point

## What NativeBot is NOT

- Not a cloud platform or SaaS
- Not an IDE or code editor (use VS Code for that)
- Not a general-purpose AI assistant
- Not a web app builder (it builds mobile apps)
- Not trying to replace Expo or React Native

## Principles

1. **Zero friction** — `pip install nativebot && nativebot` should just work
2. **No accounts** — No signup, no login, no cloud dependency
3. **Real code** — Not toy demos. Production-ready Expo apps you can ship
4. **Your code** — Files live on your filesystem. Open in any editor. Git it, fork it, deploy it
5. **AI-first** — The chat IS the interface. Not an IDE plugin

## What We Will Merge

- Bug fixes
- New Expo template improvements
- Better CLI UX (colors, prompts, progress)
- Documentation improvements
- Model support updates

## What We Won't Merge

- Web UI or dashboard
- Cloud storage or sync
- User accounts or auth
- Billing or payment features
- Non-Expo frameworks (Flutter, native Swift/Kotlin)
